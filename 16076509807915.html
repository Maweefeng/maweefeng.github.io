Unit testing

2020/12/11

添加单元测试到项目当中，然后如果想要调用项目中的方法，需要在顶部加 @testable import 工程名

一个简单的单元测试如下代码所示

import XCTest
@testable import  Appstore
class AppstoreTests: XCTestCase {
    
    func testSqure() {
        let value = 7
        let squredvalue = value.squreInt()
        XCTAssertEqual(squredvalue, 49)
    }

    func sum() {
        let a = 5
        let b = 6
        let c = a + b
        XCTAssertEqual(c, 11)
    }
}

那么如果一开始创建项目没有勾选单元测试，后期如何加单元测试到项目当中呢？

添加单元测试到项目

  • 第一步选择项目，按左下角加号添加target

  • 第二步，选择iOS Unit Testing Bundle
